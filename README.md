# Portfolio

Personal portfolio site built with Astro.

## Deployment Target

This project is intended to be deployed on GitHub Pages.

Implications:
- Keep the site compatible with static hosting.
- Commit source files and configuration, not generated output like `dist/`.
- The production build output is `dist/`, which should be generated by CI or a local build step before deployment.
- The GitHub Actions workflow is in `.github/workflows/deploy.yml`.
- `astro.config.mjs` auto-detects whether the repository is a user site (`<user>.github.io`) or a project site (`/<repo>/`) and sets the correct `base` path during CI builds.

## Commands

All commands run from the project root:

| Command | Action |
| :------ | :----- |
| `npm install` | Install dependencies |
| `npm run dev` | Start local development server |
| `npm run build` | Build the static site into `dist/` |
| `npm run preview` | Preview the production build locally |
